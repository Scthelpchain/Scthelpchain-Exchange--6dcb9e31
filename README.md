## 📖 Vision
SCTHelpChain is a global initiative led by **Habibur Rahman Antor (HR Antor)** to bridge the gap between Islamic education and modern financial technology. We are building the world's first AI-driven Madrasa ecosystem integrated with a decentralized payment gateway.

## 🚀 Key Features
* **AI Madrasa Engine:** Real-time Makhraj and Tajweed analysis using advanced voice recognition.
* **SCT Luxury Master Card:** A global crypto-fiat debit card for seamless transactions.
* **Charity Gateway:** Transparent, blockchain-based donation system for orphanages.
* **Web3 Integration:** Full MetaMask and WalletConnect support.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Blockchain:** Web3.js, Solidity
- **Hosting:** Netlify & GitHub Pages
- **AI Model:** Custom Speech-to-Text for Quranic Arabic.
